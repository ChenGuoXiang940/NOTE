# Mermaid語法設計指南
使用Mermaid語法時，最前加上一行「\`\`\`mermaid」表示程式開始，最後再加上一行「\`\`\`」表示程式結束。  
## 定義方向
以 graph 為開頭，宣告圖表方向，可以表示四種相對的方向：
* **TB/TD** 上到下
* **BT** 下到上
* **RL** 右到左
* **LR** 左到右

## 繪製流程
```mermaid
graph TD;
    程序A-->程序B;
    程序A-->程序C;
    程序B-->程序D;
    程序B-->程序E;
    程序C-->程序F;
    程序C-->程序G;
```
